# פרופיל פיטנס

## בסיס

- גיל: 21, גבר, ישראלי
- גובה: 170cm | משקל: 66.7kg (מאי 2026) → 68.4kg (יוני 2026)
- אחוז שומן: ~11–13%
- שלב: לין בולק — סיים חיתוך, עודף קלורי הדרגתי
- יעד קלורי: ~2600 קק"ל | יעד חלבון: 130–150g ליום

---

## אימוני כוח — פיצול A / B / C

**A — חזה, יד אחורית, כתפיים**
- לחיצת חזה בנץ, שיפוע 30°, פרפר, מקבילים
- Overhead Tricep Extension, Skull Crusher מוט W
- לחיצת כתפיים, Lateral Raises, Face Pull

**B — גב, יד קדמית**
- מתח עם משקל עד כשל — 6 סטים (drop set)
- Dumbbell Row, מתח רחב
- כפיפות מרפקים, Hammer Curls

**C — רגליים, ישבן, שוקיים**
- סקוואט עם מוט 4×6-10, Bulgarian Split Squat
- לחיצת רגל, Hip Thrust, RDL, עליות שוק

**לוח שבועי:** A / B / מנוחה / C / A / B / C-שישי
**כלל:** C תמיד בשישי — שבת = ריקברי מלא לרגליים
